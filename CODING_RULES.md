Coding rules:
1. Prioritize correctness over speed.
2. Keep files modular and small.
3. Do not change model choice without asking.
4. Do not modify cross-attention.
5. Do not silently change the attention formula.
6. Add comments for all tensor shapes.
7. Every script must work both locally and on Kaggle.
8. Use Hugging Face Transformers and PyTorch.
9. Add simple tests whenever possible.
10. Do not implement everything in one file.
